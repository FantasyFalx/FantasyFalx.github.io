Counterfeit Indentifier

Analyzed 1,373 rows of USD currency data to predict counterfeit notes using statistical features such as variance, skewness, kurtosis, and entropy. Developed six visualizations to compare classification attributes, providing clear insights into the relationships between features. Implemented probability analysis with Euclidean distance to enhance prediction accuracy, using color encoding to visually represent accurate predictions and errors.
